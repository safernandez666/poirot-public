Get Started in Minutes

Simple setup, powerful results. Start protecting your data in three easy steps.

1

Deploy Poirot

Run with Docker Compose. No complex installation or configuration needed. The dashboard and scanner start automatically.

2

Connect Sources

Add your data sources through the UI or environment variables. Support for credentials, IAM roles, and connection strings.

3

Start Scanning

Run your first scan and watch as Poirot discovers and classifies sensitive data across all your connected sources.
